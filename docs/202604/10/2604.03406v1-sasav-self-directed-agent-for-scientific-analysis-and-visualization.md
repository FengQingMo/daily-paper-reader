## 摘要
随着前沿多模态大型语言模型（MLLMs）在数据理解与视觉推理方面的最新进展，LLM 的角色已从被动的“LLM-作为接口”演变为主动的“LLM-作为判断者”，从而实现与科学数据分析和可视化流程的更深层次整合。然而，现有的科学可视化智能体仍依赖领域专家为特定数据集或以可视化为导向的目标函数提供先验知识，以通过迭代反馈引导工作流程。这种反应式、数据依赖的人类参与（HITL）范式既耗时，又难以有效扩展至大规模科学数据。在本研究中，我们提出了一个用于科学分析与可视化的自导式智能体（SASAV），这是首个能够在没有任何外部提示或人类反馈的情况下，完全自主执行科学数据分析并生成深刻洞察可视化的人工智能智能体。SASAV 是一个多智能体系统，可通过我们提出的组件自动协调数据探索工作流程，包括自动化数据剖析、具备上下文感知的知识检索以及基于推理驱动的可视化参数探索，同时支持下游交互式可视化任务。本研究为未来的科学领域人工智能奠定了基础模块，以加速科学发现与创新的规模化发展。

## Abstract
With recent advances in frontier multimodal large language models (MLLMs) for data understanding and visual reasoning, the role of LLMs has evolved from passive LLM-as-an-interface to proactive LLM-as-a-judge, enabling deeper integration into the scientific data analysis and visualization pipelines. However, existing scientific visualization agents still rely on domain experts to provide prior knowledge for specific datasets or visualization-oriented objective functions to guide the workflow through iterative feedback. This reactive, data-dependent, human-in-the-loop (HITL) paradigm is time-consuming and does not scale effectively to large-scale scientific data. In this work, we propose a Self-Directed Agent for Scientific Analysis and Visualization (SASAV), the first fully autonomous AI agent to perform scientific data analysis and generate insightful visualizations without any external prompting or HITL feedback. SASAV is a multi-agent system that automatically orchestrates data exploration workflows through our proposed components, including automated data profiling, context-aware knowledge retrieval, and reasoning-driven visualization parameter exploration, while supporting downstream interactive visualization tasks. This work establishes a foundational building block for the future AI for Science to accelerate scientific discovery and innovation at scale.

## 二、方法论：核心思想与技术架构

### 1. 总体架构
- SASAV 是一个多智能体系统（multi-agent system），结合 MLLM 的视觉–语言推理能力，实现：
  1. **数据剖析（Data Profiling）**  
     自动识别数据格式、结构、类型及初步视角。
  2. **知识检索（Knowledge Retrieval）**  
     从网络与本地知识库（RAG）获取上下文科学信息，定位具有科学意义的“感兴趣区域”。
  3. **传输函数建议（Transfer Function Suggestion）**  
     自动推荐颜色与不透明度映射，使视觉上突出科学意义区域。
  4. **视角选择（View Selection）**  
     自动选择最佳观察角度及探索轨迹，生成静态图像与动画。

### 2. 架构细节与算法流程

- **Data Profiling阶段**
  - 提取元数据（维度、间距、类型等）并进行降采样处理。
  - 初次渲染采用白黑X射线风格以保留结构梯度。
  - 定义不透明度传输函数（Ramp-shaped TF）：

    \[
    \text{Opacity}(v) = 
    \begin{cases} 
    0, & v < RSV \\[3pt]
    \frac{v - RSV}{V_{\max} - RSV}, & v \ge RSV
    \end{cases}
    \]

    其中 RSV 通过均匀采样 [Vmin, Vmax] 多取值以应对噪声差异。

  - 利用多视角（六个正交视角）进行初步渲染 → MLLM Evaluator评分 → 选出最佳TF与视角。

- **Knowledge Retrieval阶段**
  - 通过 *Forager Agent* 结合两种检索：
    - 网络搜索（OpenAI Agents SDK）
    - 本地 RAG 向量数据库（LangChain + Chroma + text-embedding-3-large 模型）
  - 抽取科学关键词并排序，用于后续TF与视角决策。

- **Transfer Function Suggestion阶段**
  - 区分两类数据：
    - 模拟数据：采用连续DVR（Direct Volume Rendering）
    - 经验数据：采用离散Isosurface Rendering
  - 设计双智能体：
    - **Semantic Analyzer (SA)**：对每个取样等值面多视角渲染图进行打分（科学显著度、遮挡风险、信心等）。
    - **Transfer Function Designer (TFD)**：汇总SA结果，生成颜色和不透明度曲线。对于经验数据，还包括：
      - *Isovalue Rejection*（剔除低信心等值面）
      - *Isovalue Fine-Tuning (IFT)*：通过比较式推理调整等值面位置以增强可辨识度。

- **View Selection阶段**
  - 在包络球面上采样视角（K=32，Fibonacci lattice）。
  - 使用MLLM评估各视角的可视化质量、信息量与遮挡情况。
  - 输出：
    - 信息量排序视图集合
    - Anchor视图序列用于轨迹生成（Catmull–Rom样条插值）

### 3. 实现技术要点
- 使用低温（temperature=0.1）以保证输出稳定。
- 多线程和并行MLLM调用降低推理时延。
- 中间数据均使用规范化JSON传递。
- 可视化生成通过 C++ + VTK 实现，本地和分布式渲染均支持。
