## 摘要
当自主智能体分别观察共享环境的不同领域时，它们交换的每个信号都会混合相关与无关的维度。目前没有机制能让接收方评估哪些维度值得吸收。我们提出了符号-向量注意力融合（Symbolic-Vector Attention Fusion，SVAF），这是一个用于群体智能的两层耦合引擎中的内容评估部分。SVAF将每个智能体间的信号分解为7种有类型的语义域，通过学习得到的融合门对每个语义域进行评估，并生成一个重新混合的结果——从两个领域的交集中产生的新知识。一个带通模型产生四种结果（冗余、对齐、防护、拒绝），从而同时解决选择性与冗余问题。融合门能够独立发现跨领域相关性层级：在第1个epoch时，“情绪”便成为权重最高的语义域，早于准确度的稳定期——这与独立的机械证据一致，证明LLM的情绪表示在价度-唤醒轴上具有结构嵌入特性。SVAF构成网状记忆协议（Mesh Memory Protocol，MMP）的第4层；耦合引擎的另一部分是第6层的每智能体闭式连续时间（Closed-form Continuous-time，CfC）神经网络，其学习到的每个神经元的时间常数（tau）创造了群体智能产生的时间动态：快速神经元在数秒内跨智能体同步情感，而慢速神经元能无限期地保持领域专长。SVAF决定进入每个智能体认知状态的内容；CfC决定该状态的演化。在273个叙事场景的23.7万个样本上训练后，SVAF达到78.7%的三分类准确率。我们在macOS、iOS和Web上的7节点实时部署中验证了完整的网状认知循环——从语义域评估到重混、CfC状态演化、tau调制的同伴融合以及自主行动。

## Abstract
When autonomous agents observe different domains of a shared environment, each signal they exchange mixes relevant and irrelevant dimensions. No existing mechanism lets the receiver evaluate which dimensions to absorb. We introduce Symbolic-Vector Attention Fusion (SVAF), the content-evaluation half of a two-level coupling engine for collective intelligence. SVAF decomposes each inter-agent signal into 7 typed semantic fields, evaluates each through a learned fusion gate, and produces a remix -- new knowledge from the intersection of two domains. A band-pass model yields four outcomes (redundant, aligned, guarded, rejected), solving both selectivity and redundancy. The fusion gate independently discovers a cross-domain relevance hierarchy: mood emerges as the highest-weight field by epoch 1, before accuracy plateaus -- consistent with independent mechanistic evidence that LLM emotion representations are structurally embedded along valence-arousal axes. SVAF forms Layer 4 of the Mesh Memory Protocol (MMP); the other half of the coupling engine is a per-agent Closed-form Continuous-time (CfC) neural network at Layer 6, whose learned per-neuron time constants (tau) create the temporal dynamics from which collective intelligence emerges: fast neurons synchronise affect across agents in seconds, while slow neurons preserve domain expertise indefinitely. SVAF determines what enters each agent's cognitive state; CfC determines how that state evolves. Trained on 237K samples from 273 narrative scenarios, SVAF achieves 78.7% three-class accuracy. We verify the complete mesh cognition loop -- from per-field evaluation through remix, CfC state evolution, tau-modulated peer blending, and autonomous action -- in a live deployment with 7 nodes across macOS, iOS, and web.

## 一、研究问题与背景

- **核心问题**：在多智能体系统中，不同领域的自主智能体观察同一环境的不同方面，交换的信息往往包含“有用”与“无关”的混合信号。目前缺少机制可让接收者在信号层面判断哪些信息值得吸收。
- **目标**：设计一种机制，使多智能体间能在保持领域独立性的同时，高效吸收跨域有价值的信息，实现“**选择性吸收 + 冗余检测**”。
- **研究意义**：
  - 打破以往单向“交流－存储”框架，转而关注“内容是否值得吸收”的跨域评估问题。
  - 支撑集体智能（collective intelligence）的形成，即多个异构智能体在共享环境中演化出协同认知能力。

## 二、方法论：Symbolic-Vector Attention Fusion（SVAF）

### 1. 整体框架

- SVAF 是 **Mesh Memory Protocol (MMP)** 的第4层机制（内容评估层），决定“什么信息进入智能体的认知状态”。
- 与之配合的是第6层的 **Closed-form Continuous-time（CfC）网络**，控制“认知状态如何随时间演化”。
- 两者共同构成群体智能的“双层耦合引擎”。

### 2. 核心思想

- 将任意跨域信号分解为 **7个固定语义域（CAT7）**，每个域以“符号（文本）+向量”双重形式表示：
  | 语义域 | 含义 | 跨域相关性 |
  | :-- | :-- | :-- |
  | focus（焦点） | 信息主题 | 中 |
  | issue（问题） | 风险或张力 | 中 |
  | intent（意图） | 目标或期望 | 低 |
  | motivation（动机） | 原因与驱动力 | 中 |
  | commitment（承诺） | 任务与执行 | 中 |
  | perspective（视角） | 观察者背景 | 低 |
  | mood（情绪） | 价度-唤醒状态 | 高（跨域） |

- **信号评估流程**：
  1. **编码 (Encode)**：各域文本通过共享主干 + 域专属投影头编码成64维语义向量。
  2. **语义域间注意力 (Contextualise)**：7个域间通过多头注意力机制交互，捕捉内部依赖。
  3. **评估 (Evaluate)**：对比接收智能体自身的“锚定记忆”（anchor），计算语义漂移（drift）。
  4. **融合 (Synthesise)**：通过**学习的融合门（fusion gate）**与非线性转换生成“重混”结果（remix）。
- **融合门机制**：
  - 门值 \( g_f ∈ [0,1] \) 控制每个语义域吸收强度；
  - 高门值代表信号域被保留，低门值则由本地上下文替代；
  - 评估输出包括四种状态：**冗余、对齐、防护、拒绝**。
- **时间衰减模型**：不同领域有独立的时间窗口τ，用于加权信号的新鲜度。
- **数学核心**（文字描述）：
  - 通过余弦相似度计算每个语义域间漂移；
  - 汇总得到总体漂移，再经带通阈值决定信号类别；
  - 冗余：全部域低漂移；对齐：总体低漂移；防护：中漂移；拒绝：高漂移。

### 3. 神经拓展（Neural SVAF）

- 引入 **跨域注意力 + 神经门控层**：
  - Anchor attention 融合历史上下文；
  - 神经网络生成7维门向量 \( g \)；
  - MLP 的非线性混合代替线性权重平均，实现复杂融合动力。
- **训练目标** 由四项组成：
  1. 三分类交叉熵（对齐/防护/拒绝）；
  2. 漂移预测的MSE误差；
  3. 门方向正则（拒绝样本门应低）；
  4. 情绪优先约束（情绪门值应高于其他域）。

## 三、实验设计

### 1. 数据与场景

- **数据来源**：共 273 条 LLM 创作的多智能体叙事场景（Narratives），共 237,120 个样本。
  - 每个叙事含 4–10 条时序信号，描述如“晨间倦怠”“运动恢复”等人机交互过程。
- **样本生成**：
  - 每条信号与所有潜在接收智能体配对；
  - 由领域规则生成“有用性”与“新鲜度”标签；
  - 输出决策标签（对齐/防护/拒绝）。
- **训练与验证划分**：219 场景(18.8万样本)训练，54 场景(4.8万样本)验证，无叙事泄漏。

### 2. 对比方法（基线）

| 方法 | 描述 | 特征 |
| :-- | :-- | :-- |
| Scalar | 整体信号单向量余弦相似度 | 无领域区分 |
| Scalar + Temporal | 加入时间衰减 | 简单时序加权 |
| Heuristic per-field | 各域独立余弦 + αf加权 | 半启发式 |
| **SVAF (neural)** | 全模型（注意力+融合门） | 本文提出 |

- **性能指标**：三分类准确率（Aligned/Guarded/Rejected），以及F1、Precision、Recall。

## 六、主要结论与发现

1. **准确率提升**：SVAF（神经版）三分类总体准确率 **78.7%**，较基线提高约 12 个百分点。  
2. **情绪维度的自动涌现**：融合门自动学习到“情绪”为最具跨域相关性的语义域，显著高于其他域（0.497 vs 0.056–0.295）。  
3. **两层耦合架构有效**：SVAF 栅门评估（内容层）与 CfC 网络（时间演化层）联合实现群体智能闭环。  
4. **真实部署验证**：多设备节点中，智能体间认知漂移从 0.936→0.468 在一循环内收敛，证明机制可自启动协作。  
5. **系统发布**：开源实现（Node.js 与 Swift），并提供完整协议规范。

## 八、局限与不足

- **数据合成偏差**：训练标签由规则生成，缺少人工标注，可能导致模型学习到任务构造偏好而非真实相关性。  
- **实验范围有限**：验证仅覆盖单用户多代理应用，未展示多用户、多域竞争场景下的长周期行为。  
- **模型监督强度不足**：门控学习仅通过软约束（非显式标签），对更复杂领域关系的学习能力尚未验证。  
- **编码质量瓶颈**：启发式路径仍依赖简化n-gram编码，限制语义一致性；需高质量语义嵌入才能发挥SVAF潜能。  
- **算力与实时性平衡**：神经版推理延迟高（冷启动约5.8秒），目前生产主要依赖启发式快速路径。
