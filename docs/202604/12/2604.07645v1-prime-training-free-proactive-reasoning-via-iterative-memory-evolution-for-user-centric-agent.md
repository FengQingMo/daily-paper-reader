## 摘要
为复杂、长周期任务开发能够与人类用户协作的自主工具使用代理，已成为智能体研究的前沿方向。在多轮人机交互过程中，用户需求的动态性和不确定性带来了重大挑战；智能体不仅需要调用工具，还必须通过有效沟通不断完善对用户意图的理解。尽管近期强化学习的进展为构建更强大的工具使用代理提供了可能途径，但现有方法依赖昂贵的训练成本，并在长时交互中难以实现逐轮的信用分配。为此，我们提出了 PRIME（Proactive Reasoning via Iterative Memory Evolution），一种无梯度学习框架，通过显式经验积累而非昂贵的参数优化，实现智能体的持续进化。PRIME 将多轮交互轨迹提炼为结构化、可被人类理解的经验，并按三种语义区域组织：成功策略、失败模式和用户偏好。这些经验通过元层次操作不断演化，并通过检索增强生成来引导未来的智能体行为。我们在多个不同的以用户为中心的环境中进行实验，结果显示 PRIME 在保持成本效率和可解释性的同时，取得了与基于梯度的方法相当的性能。总体而言，PRIME 为构建一种能够从人机交互中学习、且无需梯度训练计算负担的主动协作智能体提供了一种切实可行的范式。

## Abstract
The development of autonomous tool-use agents for complex, long-horizon tasks in collaboration with human users has become the frontier of agentic research. During multi-turn Human-AI interactions, the dynamic and uncertain nature of user demands poses a significant challenge; agents must not only invoke tools but also iteratively refine their understanding of user intent through effective communication. While recent advances in reinforcement learning offer a path to more capable tool-use agents, existing approaches require expensive training costs and struggle with turn-level credit assignment across extended interaction horizons. To this end, we introduce PRIME (Proactive Reasoning via Iterative Memory Evolution), a gradient-free learning framework that enables continuous agent evolvement through explicit experience accumulation rather than expensive parameter optimization. PRIME distills multi-turn interaction trajectories into structured, human-readable experiences organized across three semantic zones: successful strategies, failure patterns, and user preferences. These experiences evolve through meta-level operations and guide future agent behavior via retrieval-augmented generation. Our experiments across several diverse user-centric environments demonstrate that PRIME achieves competitive performance with gradient-based methods while offering cost-efficiency and interpretability. Together, PRIME presents a practical paradigm for building proactive, collaborative agents that learn from Human-AI interaction without the computational burden of gradient-based training.

## 二、方法论概述：PRIME 框架

### 核心思想
- **PRIME（Proactive Reasoning via Iterative Memory Evolution）**是一个**无梯度的学习框架**。  
  它不调整模型参数，而是通过积累多轮交互的“经验”逐步改进智能体行为。
- 将学习过程看作是**记忆库的优化问题**，而非传统参数优化。

### 关键技术机制
1. **记忆结构化（Memory Library）**  
   交互轨迹被提炼为结构化的“经验单元”：
   - *情况（situation）*：发生的场景；
   - *行动（action）*：智能体决策；
   - *结果（outcome）*；
   - *经验与教训（lesson）*；  
   并按语义类别划分为三种“区域（zone）”：  
     - **Golden Zone**：成功策略；  
     - **Warning Zone**：失败模式；  
     - **Preference Zone**：用户偏好模式。

2. **交互阶段标注（Stage-aware tagging）**  
   每段经验按互动阶段标识：
   - 探索阶段（Exploration: 信息收集）；
   - 验证阶段（Verification: 意图细化）；
   - 完成阶段（Completion: 方案输出）。

3. **四类记忆演化操作（Figure 5）**  
   - **Mutation（变异）**：根据反馈优化经验表述；
   - **Generalization（泛化）**：去除环境细节，使经验可迁移；
   - **Crossover（交叉）**：融合互补经验；
   - **Pruning（剪枝）**：删除过时或低效记忆。

4. **无梯度优化流程**
   - 交互过程中生成轨迹 τ；
   - 按奖励函数 R(τ) 执行多轮信用分配；
   - 用 LLM 进行语义提炼形成记忆；
   - 经上述演化操作更新库 M；
   - 在推理时从库中检索与当前状态相关的经验，用作增强提示（retrieval-augmented generation）。

5. **主要数学表述（文字说明）**
   - 目标：寻找最佳记忆库 \( M^* \)，最大化代理性能：
     \[
     M^* = \arg\max_M E[\Phi(\pi(\cdot|X_i, m_i), Y_i)]
     \]
     其中 π 是冻结的 LLM，Φ 衡量输出的任务完成度。
   - 记忆更新遵循迭代式分布：
     \[
     M_{i+1} \sim μ(\cdot|M_i, \{τ_i | X_i, π\})
     \]
     即通过前向演化不断更新库，不涉及反向梯度传播。

## 六、主要结论与发现

- PRIME 在所有八个环境中**均提升**基线模型性能（平均提升 20–200% 不等）。

- 框架具有**可解释性与审计性**：每个经验均可溯源、查看来源与逻辑。

## 七、方法与实验亮点

- **创新点**：
  - 摒弃梯度训练，将学习视为**显式经验积累与进化**；
  - 记忆库设计实现**结构化语义学习**（分区+分阶段标签）；
  - 通过**元层级操作**（变异、泛化、交叉、剪枝）实现持续改进；
  - 支持**跨环境、跨架构迁移**；
  - **效果可解释**、成本极低，尤其适用于部署阶段在线学习。

- **应用亮点**：
  - 在旅行、意图、逻辑游戏、数学推理等场景中均有显著效率提升；
  - 框架简易、无需模型再训练，易于系统集成。

## 八、不足与局限

- **性能上限受限**：  
  PRIME 依赖基础模型的推理质量，不能像RL那样对模型内部权重优化。
- **经验提炼依赖外部大模型（如GPT-4o）**，存在算力与访问成本。
- **经验迁移效果有衰减**：不同模型架构间存在差异，迁移库的提升幅度较低。
- **尚未验证真实人类用户交互**：实验基于模拟用户，可能与真实场景存在偏差。
- **演化操作参数手动设定**，未有自适应或理论保证最优。
