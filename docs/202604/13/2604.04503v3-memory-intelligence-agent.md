## 摘要
深度研究智能体（Deep Research Agents，DRAs）将大语言模型（LLM）的推理能力与外部工具结合。记忆系统使DRAs能够利用历史经验，这是高效推理与自主进化的关键。现有方法通常依赖于从记忆中检索相似轨迹来辅助推理，但面临记忆进化效率低下以及存储和检索成本增加的问题。为了解决这些问题，我们提出了一种新颖的记忆智能体（Memory Intelligence Agent，MIA）框架，由管理者-规划者-执行者（Manager-Planner-Executor）架构组成。Memory Manager 是一种非参数化记忆系统，可存储压缩的历史搜索轨迹；Planner 是一种参数化记忆智能体，能够为问题生成搜索计划；Executor 则是另一个根据搜索计划执行搜索与信息分析的智能体。为了构建 MIA 框架，我们首先采用交替强化学习范式，以增强 Planner 与 Executor 之间的协作。此外，我们使 Planner 在测试阶段学习中能够持续进化，在推理过程不中断的情况下即时更新。与此同时，我们在参数化与非参数化记忆之间建立了双向转换环路，实现高效的记忆进化。最后，我们引入反思机制和无监督判断机制，以增强在开放环境中的推理与自我进化能力。在十一项基准测试上的大量实验结果表明，MIA 具有显著的优越性。

## Abstract
Deep research agents (DRAs) integrate LLM reasoning with external tools. Memory systems enable DRAs to leverage historical experiences, which are essential for efficient reasoning and autonomous evolution. Existing methods rely on retrieving similar trajectories from memory to aid reasoning, while suffering from key limitations of ineffective memory evolution and increasing storage and retrieval costs. To address these problems, we propose a novel Memory Intelligence Agent (MIA) framework, consisting of a Manager-Planner-Executor architecture. Memory Manager is a non-parametric memory system that can store compressed historical search trajectories. Planner is a parametric memory agent that can produce search plans for questions. Executor is another agent that can search and analyze information guided by the search plan. To build the MIA framework, we first adopt an alternating reinforcement learning paradigm to enhance cooperation between the Planner and the Executor. Furthermore, we enable the Planner to continuously evolve during test-time learning, with updates performed on-the-fly alongside inference without interrupting the reasoning process. Additionally, we establish a bidirectional conversion loop between parametric and non-parametric memories to achieve efficient memory evolution. Finally, we incorporate a reflection and an unsupervised judgment mechanisms to boost reasoning and self-evolution in the open world. Extensive experiments across eleven benchmarks demonstrate the superiority of MIA.

### 1. 研究动机  
- **深度研究智能体（Deep Research Agents, DRAs）** 旨在结合大语言模型（LLM）的逻辑推理能力与外部搜索、工具调用，以完成复杂、多轮次的开放式推理任务。  
- 随着智能体任务复杂度和交互轮次增加，**记忆系统** 成为关键：它影响智能体是否能利用历史经验、提炼搜索策略，实现“学习-反思-进化”的长程智能。
- 当前基于 **长上下文（long-context）记忆** 的方法存在以下问题：
  - 注意力被稀释，模型难以聚焦于当前任务。  
  - 冗余或噪声信息被持续积累，降低推理稳定性。  
  - 存储和检索成本随时间线性上升。  
  - 记忆多为“结果性”知识，缺乏对“过程性”搜索轨迹的利用。
- 因此，论文聚焦于解决：
  1. **记忆演化低效**——难以从历史推理中持续学习。  
  2. **存储/检索成本高**——传统记忆扩展造成推理延迟和成本上升。

## 二、方法论：MIA 框架设计与技术实现

### 1. 整体结构：Manager–Planner–Executor 架构
- **Memory Manager（记忆管理者）**  
  - 非参数化记忆模块，用于存储压缩后的历史轨迹（成功与失败实例）。  
  - 采用三维度检索：**语义相似度、价值奖励、频次奖励**，以平衡相关性与多样性。
- **Planner（规划者）**  
  - 参数化智能体，负责生成解题计划。  
  - 在推理过程中可调用记忆案例指导规划，同时在反思阶段进行自我修正。  
- **Executor（执行者）**  
  - 根据 Planner 提供的计划执行多步推理与外部信息检索（支持多模态搜索）。
- 三者形成一个 **“记忆检索–计划生成–执行反馈–反思更新”循环**，实现代理的自主学习与记忆优化。

### 2. 关键机制
- **交替强化学习（Alternating Reinforcement Learning, RL）**
  - 分两阶段交替训练：
    1. 冻结 Planner，训练 Executor 理解/执行计划；
    2. 冻结 Executor，训练 Planner 在带有记忆上下文条件下优化规划与反思。
  - 均采用 **Group Relative Policy Optimization (GRPO)** 算法进行策略更新。
- **经验压缩与结构化记忆**  
  - 使用大模型（如 Qwen3-32B）生成图像字幕与工作流摘要，将原始搜索轨迹压缩为结构化条目，节省存储。  
- **测试时学习（Test-Time Learning, TTL）**
  - 在推理过程中同步进行在线更新：
    - 非参数化记忆：提炼成功与失败的路径形成对比记忆。  
    - 参数化记忆：实时更新 Planner 的参数，使其持续优化规划策略。
- **无监督自评估机制（Peer-review Evaluation Framework）**
  - 模拟学术评审体系，采用三个虚拟审稿者 + 一名“领域主席”：
    - 从逻辑一致性、信息可信度、结果完整性三个维度独立评估，
    - 最终整合判断用于在无标注环境中驱动自进化。
